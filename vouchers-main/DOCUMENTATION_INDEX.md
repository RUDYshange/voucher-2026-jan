(Important!)\n\nAlways remember:\n\n| Role | Percentage | Amount |\n|------|-----------|--------|\n| **Buyer Pays** | 90% | R900 (for R1000 voucher) |\n| **Seller Gets** | 70% | R700 (for R1000 voucher) |\n| **Platform Fee** | 20% | R200 (for R1000 voucher) |\n\nExample calculation in code:\n```typescript\nconst faceValue = 1000\nconst buyerPrice = faceValue * 0.9   // = 900\nconst sellerPayout = faceValue * 0.7 // = 700\nconst platformFee = faceValue * 0.2  // = 200\n```\n\n**See**: [DOCUMENTATION.md](./DOCUMENTATION.md#pricing-model) for more details\n\n---\n\n## 🔧 Technology
